平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示GPIO模块Pin的输入功能。

说明：
本样例通过读取按键K10状态控制GPIO引脚输出高低电平，驱动LED亮灭。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F4A0_LQ176_Rev1.0

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序并运行；
3）按K10按键，观察开发板上的LED11：当按下K10时，LED11亮；松开K10后，LED11灭。
