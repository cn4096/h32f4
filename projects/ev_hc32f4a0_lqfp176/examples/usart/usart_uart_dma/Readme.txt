平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示USART外设配置为UART模式时通过定时器TMR0配合，实现不定长度数据接收。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F4A0_LQ176_Rev1.0

辅助工具:
---------------------
无

辅助软件:
---------------------
串口助手软件

================================================================================
使用步骤
================================================================================
1）通过USB数据线，连接PC和板子DAP
2）打开串口助手软件，配置端口如下参数
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
3）打开工程并重新编译
4）下载代码测试板，并且全速运行
5）串口助手软件窗口输入字母或数字（数据长度不超过500字节），点击发送
6）串口助手接收窗口回显输入内容
7）重复步骤5和6

================================================================================
注意
================================================================================
1）由于接收缓存空间限制，一次可接收最大数据长度为500字节；
2）由于数据长度不固定，通过定时超时判断数据传输是否完成，所以当产生接收超时后，
   判断为数据接收完成，回发数据。
